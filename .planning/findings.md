## 二、学习指南核心要点

```
第一阶段：C# → 多线程 → SQL
```

## 四、技术原理记录

### WinForm动态控件创建（5/29）
- new → 设属性 → Controls.Add
- 类级别变量才能在事件方法中访问
- sender必须转型

### Panel/GroupBox容器（5/30）
- 容器内控件坐标是相对容器左上角
- GroupBox标题占用顶部空间
- 容器尺寸 = 控件宽度 + 边距
