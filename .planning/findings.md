# 研究发现

> 目标：2027年3月入职成都上位机初级岗
> 核心结论：初级岗 ≠ 全家桶，能连设备 + 不卡UI + 存数据 = 能上班

---

## 初级岗面试官到底看什么（6份成都/深圳JD + 自行判断）

| 面试官问 | 你要会 | 权重 |
|------|------|:--:|
| 能连上设备吗？ | TCP / SerialPort / Modbus 读写 | 🔴 |
| UI 会卡吗？ | Thread + Invoke 跨线程 | 🔴 |
| 掉线了怎么办？ | 断线重连 + 超时 | 🔴 |
| 数据存了吗？ | SQL Server + Dapper | 🟡 |
| WPF 会用吗？ | MVVM + DataGrid + 曲线 | 🟡 |
| 做过项目吗？ | 一个能演示的小项目 | 🔴 |

## 真正不需要的（比列需要什么更重要）

| 不学 | 理由 |
|------|------|
| EF Core | 工控机扛不住，初级岗不要求 |
| OPC UA | 初级岗不碰 |
| Prism/IOC | 中级框架，不是初级该想的 |
| WPF 动画/ControlTemplate | 够用即停 |
| 多协议架构 | 那是中级岗的事 |
| SynchronizationContext | Invoke 一种吃透就够 |
| ConcurrentQueue | 面试能说就行，不写 |
| MySQL 深钻 | 面试不考，概念了解即可 |
