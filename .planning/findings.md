# 研究发现

> 目标：2027年3月 成都上位机初级岗 7-10k

---

## 路线决策

| 决策 | 理由 |
|------|------|
| 先 SQL 再 WinForms | 数据是地基，先学会存和查 |
| WinForms 再 WPF | 先出成品找信心，再升级界面 |
| 多线程放通信前面 | 通信必开线程，不懂多线程没法写 |
| 运动控制最后 | 覆膜机调试经验直接平移，是核心竞争力 |
| 视觉只了解 | 上位机岗不需要深究视觉 |

## 资源

- SQL：SSMS + 微软文档
- WinForms：茗工-JGTech B站
- 多线程：刘铁猛 + 微软文档
- 通信：HslCommunication 官网
- WPF：社区教程 + CommunityToolkit
- 运动控制：固高/雷赛 SDK 文档
- 视觉：Halcon B站 + AI 辅助
